서울과학기술대학교 경영대학 리뉴얼

📅 2025년 1월 👤 역할: UI/UX 퍼블리셔

프로젝트 개요

서울과학기술대학교 경영대학 웹사이트를 전면 리뉴얼하며, 다양한 디바이스 환경에서 일관된 사용자 경험을 제공하기 위해 반응형 구조로 개선한 프로젝트입니다.

특히 웹접근성 기준을 충족할 수 있도록 마크업 구조와 코드 품질에 중점을 두고 작업하여, 최종적으로 웹접근성 검사 통과 성과를 달성했습니다.

주요 역할 및 업무

• 반응형 구조 설계 및 구현
화면 크기에 따라 유연하게 변화하는 그리드 기반 레이아웃을 구현해 PC/태블릿/모바일 전 구간에서 일관된 사용자 경험 제공

• 웹접근성 중심 마크업 개선
시맨틱 구조와 코드 품질을 정비해 접근성 기준을 충족하고, 실제 웹접근성 검사 통과를 목표로 구조를 고도화

• 네비게이션 및 정보 구조 재정리
모바일 환경에서도 사용이 편리한 직관적 네비게이션으로 개선하고, 학과 정보/교육과정/뉴스 등 주요 콘텐츠가 명확히 전달되도록 정보 구조(IA) 재설계

• 성능 최적화
CSS 최소화와 이미지 최적화를 적용해 로딩 속도와 페이지 응답성을 개선

기술 스택

HTML5 CSS3 JavaScript CMS Figma

주요 코드 미리보기

※ 실제 운영 파일을 그대로 공개하는 대신, 핵심 구현 방식을 일반화한 예시 코드입니다. (기관명·도메인·내부 경로는 대표 예시로 대체)

1) 현재 위치와 하위 메뉴 상태를 전달하는 GNB

PC GNB는 nav와 목록 구조로 메뉴의 범위를 명확히 하고, 현재 페이지는 aria-current로 표시했습니다. 하위 메뉴를 여닫는 항목은 button으로 구성해 키보드 조작 대상임을 분명히 하고, aria-expanded와 aria-controls로 메뉴의 열림 상태와 연결 대상을 함께 전달했습니다.

<nav class="gnb" aria-label="경영대학 주요 메뉴">
    <ul class="dep" role="list">
        <li class="dep1">
            <button type="button"
                    class="a1"
                    aria-current="page"
                    aria-expanded="true"
                    aria-controls="gnb-introduction">
                학과소개
            </button>
            <ul id="gnb-introduction" class="d2" aria-label="학과소개 하위 메뉴">
                <li><a class="a2" href="/d_introduction/dean_message">학장 인사말</a></li>
                <li><a class="a2" href="/d_introduction/d_intro">학과소개</a></li>
                <li><a class="a2" href="/d_introduction/prof_intro">교수소개</a></li>
            </ul>
        </li>
    </ul>
</nav>

2) 키보드 접근성과 상태 동기화를 고려한 모바일 메뉴

모바일 메뉴는 이미지 클릭 영역이 아니라 button 요소로 구성해 기본 키보드 접근성을 확보했습니다. 클릭 시 aria-expanded와 hidden 값을 함께 갱신해, 화면에 보이는 메뉴 상태와 스크린리더가 인식하는 상태가 어긋나지 않도록 처리했습니다.

<header id="m_head">
    <h1 class="logo">
        <a href="/"><img src="/assets/img/m_logo.png" alt="로고"></a>
    </h1>
    <button type="button"
            class="m-menu-button"
            aria-controls="mobile-menu"
            aria-expanded="false">
        <span class="sr-only">전체 메뉴 열기</span>
    </button>
    <nav id="mobile-menu" class="m_menu" aria-label="모바일 전체 메뉴" hidden>
        <a class="a1" href="/d_introduction">학과소개</a>
        <a class="a2" href="/d_introduction/prof_intro">교수소개</a>
    </nav>
</header>

<script>
const menuButton = document.querySelector('.m-menu-button');
const mobileMenu = document.querySelector('#mobile-menu');

menuButton.addEventListener('click', () => {
    const isOpen = menuButton.getAttribute('aria-expanded') === 'true';
    menuButton.setAttribute('aria-expanded', String(!isOpen));
    mobileMenu.hidden = isOpen;
});
</script>

주요 성과

✓ 웹접근성 기준을 충족하는 마크업/코드 품질 개선으로 웹접근성 인증 마크 획득
✓ 반응형 그리드 레이아웃 적용으로 디바이스 전 구간에서 일관된 사용성 확보
✓ 모바일 중심 네비게이션 개선으로 탐색 편의성과 정보 접근성 향상
✓ 콘텐츠 구조 재정비로 학과 정보 전달력 강화
✓ CSS 최소화 및 이미지 최적화로 로딩 성능 개선
