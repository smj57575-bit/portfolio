서울과학기술대학교 경영대학 리뉴얼

📅 2025년 1월 👤 역할: UI/UX 퍼블리셔

프로젝트 개요

서울과학기술대학교 경영대학 웹사이트를 전면 리뉴얼하며, 다양한 디바이스 환경에서 일관된 사용자 경험을 제공하기 위해 반응형 구조로 개선한 프로젝트입니다.

특히 웹접근성 기준을 충족할 수 있도록 마크업 구조와 코드 품질에 중점을 두고 작업하여, 최종적으로 웹접근성 검사 통과 성과를 달성했습니다.

주요 역할 및 업무

• 반응형 구조 설계 및 구현
화면 크기에 따라 유연하게 변화하는 그리드 기반 레이아웃을 구현해 PC/태블릿/모바일 전 구간에서 일관된 사용자 경험 제공

• 웹접근성 중심 마크업 개선
시맨틱 구조와 코드 품질을 정비해 접근성 기준을 충족하고, 실제 웹접근성 검사 통과를 목표로 구조를 고도화

• 네비게이션 및 정보 구조 재정리
모바일 환경에서도 사용이 편리한 직관적 네비게이션으로 개선하고, 학과 정보/교육과정/뉴스 등 주요 콘텐츠가 명확히 전달되도록 정보 구조(IA) 재설계

• 성능 최적화
CSS 최소화와 이미지 최적화를 적용해 로딩 속도와 페이지 응답성을 개선

기술 스택

HTML5 CSS3 JavaScript CMS Figma

✓ 반응형 그리드 레이아웃 적용으로 디바이스 전 구간에서 일관된 사용성 확보
✓ 모바일 중심 네비게이션 개선으로 탐색 편의성과 정보 접근성 향상
✓ 콘텐츠 구조 재정비로 학과 정보 전달력 강화
