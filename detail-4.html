한동대학교 발전기금 리뉴얼

📅 2025년 3월 ~ 4월 👤 역할: UI/UX 퍼블리셔

프로젝트 개요

한동대학교 발전기금 홈페이지(Sarang Handong)를 리뉴얼하며, 기존 CMS 기반으로 운영되던 레거시 코드를 분석하고 개선해 반응형 구조로 재구성한 프로젝트입니다.

주요 역할 및 업무

• 레거시 코드 분석 및 구조 개선
기존 CMS 코드를 면밀히 분석해 중복 코드와 비효율 구간을 정리하고, CSS 통합과 시맨틱 마크업 적용으로 유지보수성을 향상

• 반응형 구조 재구성
기존 CMS 환경과의 완전한 호환성을 유지하면서도 다양한 디바이스에서 일관된 사용자 경험을 제공할 수 있도록 유연한 반응형 레이아웃 구현

• 웹접근성 개선
접근성을 고려한 마크업 및 코드 작성으로 사용자 접근성과 사용 편의성 강화

• 성능 최적화
번들 크기 감소, CSS 정리, 이미지 최적화를 통해 로딩 속도와 페이지 응답성 개선

기술 스택

HTML5 CSS3 JavaScript CMS Figma

주요 코드 미리보기

1) 반복 캠페인 카드를 의미 있는 목록 구조로 재정리

반복되는 캠페인 콘텐츠를 단순 div 묶음이 아닌 목록과 article 구조로 정리해, 스크린리더가 “캠페인 목록 안의 개별 카드”로 이해할 수 있도록 개선했습니다. 제목과 설명은 aria-labelledby/aria-describedby로 카드의 핵심 정보와 연결했습니다.

<section class="campaign-section" aria-labelledby="campaign-title">
    <h2 id="campaign-title">모금 캠페인</h2>
    <ul class="campaign-list">
        <li>
            <article class="campaign-card"
                     aria-labelledby="campaign-reed-title"
                     aria-describedby="campaign-reed-desc">
                <img src="/site/sarang/res/img/link_img01.png" alt="">
                <h3 id="campaign-reed-title">갈대상자(법인지정)</h3>
                <p id="campaign-reed-desc">기독 인재 양성을 위한 지정기부금 캠페인입니다.</p>
                <a href="/campaign/develop/reed/"
                   aria-label="갈대상자 법인지정 캠페인 자세히 보기">자세히보기</a>
            </article>
        </li>
    </ul>
</section>

2) 게시판 정보 구조와 작성일을 명확하게 제공

공지 영역은 section과 h2를 연결해 영역의 목적을 명확히 하고, 게시일은 time datetime으로 작성해 기계가 읽을 수 있는 날짜 정보로 정리했습니다. 반복되는 “더보기” 링크는 aria-label을 별도로 제공해 링크만 탐색해도 목적을 알 수 있게 했습니다.

<section class="notice-section" aria-labelledby="notice-title">
    <h2 id="notice-title">알려드립니다</h2>
    <p>기부 후원 관련 안내를 알려드립니다</p>
    <ul class="notice-list">
        <li>
            <a href="/news/notice/?mode=view&id=51674">
                한동대를 크리스천 플랫폼으로, 박성진 총장님을 소개합니다
            </a>
            <time datetime="2026-04-14">
                <span class="sr-only">작성일</span>2026.04.14
            </time>
        </li>
    </ul>
    <a class="more-link" href="/news/notice/"
       aria-label="알려드립니다 공지사항 더보기">더보기</a>
</section>
