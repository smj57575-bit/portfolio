프로젝트 개요

AI Smart 스타트업의 기업 홈페이지를 기획부터 디자인, 퍼블리싱까지 전 과정을 단독으로 수행한 프로젝트입니다.

AI 기술 기반 기업의 특성을 반영해 기술력 전달과 신뢰감 형성을 동시에 만족시키는 모던하고 직관적인 UI 구조로 설계했으며, 브랜드와 서비스 메시지가 명확하게 전달되는 정보 구조에 집중했습니다.

주요 역할 및 업무

• 기획 및 콘텐츠 전략 수립
초기 단계에서 타겟 사용자와 기업 니즈를 분석해 전체 사이트 구조를 설계하고, 자료실/공지사항 영역을 강화해 필요한 정보에 빠르게 접근할 수 있는 정보 중심 구조를 구성

• UI/UX 디자인 시스템 구축
Figma로 와이어프레임부터 고해상도 시안까지 직접 제작하고, 브랜드 톤앤매너에 맞는 일관된 디자인 시스템을 정의

• 반응형 퍼블리싱
HTML/CSS 기반으로 PC와 모바일 전 구간에서 자연스럽게 동작하는 반응형 레이아웃 구현

• 인터랙션 및 기능 구현
Cursor AI를 활용해 마크업 구조와 JavaScript 로직을 효율적으로 설계하고, 부드러운 스크롤 인터랙션, 애니메이션, 폼 검증 기능을 구현해 사용자 경험 개선

기술 스택

HTML5 CSS3 JavaScript Figma

✓ 기술력 전달과 신뢰 형성을 동시에 고려한 모던 UI로 브랜드 메시지 전달력 강화
✓ 자료실/공지사항 중심 정보 구조 고도화로 콘텐츠 접근성 및 탐색 효율 개선
✓ 반응형 구조, 인터랙션, 폼 검증을 결합해 PC/모바일 사용자 경험 향상
